# seven-languages-in-seven-weeks
Practice and exercises from Bruce Tate's book: [Seven Languages in Seven Weeks](https://amazon.com/Seven-Languages-Weeks-Programming-Programmers/dp/193435659X).
Covers seven programming languages:
  * Ruby
  * Io
  * Prolog
  * Scala
  * Erlang
  * Clojure
  * Haskell
